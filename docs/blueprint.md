# **App Name**: HRMS Lite

## Core Features:

- Add Employee: Allow admins to add new employees with details like full name, address, and department.
- View Employee List: Display a list of all employees, with options to view and delete employee records.
- Attendance Marking: Enable admins to mark attendance (present/absent) for employees on a given date.
- View Attendance Records: Provide a view to display attendance records for each employee.
- Intelligent HR Assistant: AI tool which assists HR in summarizing employee data and identifying key trends. Requires admin approval prior to integrating trends to any employee records.

## Style Guidelines:

- Primary color: Deep sky blue (#42A5F5) to give a modern and trustworthy impression, given the HR context.
- Background color: Light-blue gray (#ECEFF1).
- Accent color: Slate blue (#64B5F6) to draw attention to interactive elements.
- Body and headline font: 'Inter', sans-serif, for a modern, neutral and objective look that fits the HR application theme. This single font can be used for headlines or body text.
- Consistent use of simple, clear icons for navigation and actions.
- Clean layout with consistent spacing and intuitive navigation.
- Subtle animations for loading states and user interactions.
